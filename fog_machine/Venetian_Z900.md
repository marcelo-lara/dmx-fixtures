## 3. Filling the Fluid Tank

1. Remove the hose from the tank cap.
2. Pour fog fluid into the tank or container.
3. Close the cap tightly after filling.
4. If your model allows it, you may remove the storage tank and fill it separately to avoid spills.

Important:
- Always use suitable fog fluid.
- Avoid spilling liquid inside the machine.

---

## 4. Safety Instructions

1. Make sure the machine is unplugged before filling the fluid tank.
2. Do not allow liquid to enter the inside of the machine, as this may cause a short circuit.
3. Before connecting the power cable, confirm that the local mains voltage matches the machine rating.
4. If the factory-installed power cable is damaged, contact your local distributor.
5. If replacement is necessary, use a cable with the same characteristics.
6. When connecting the machine to power, the yellow/green ground wire must be connected correctly.
7. Keep the unit away from humidity and moisture.
8. Disconnect power immediately if the machine behaves abnormally.

## 6. Operation

1. Connect the power cable.
2. Turn on the machine using the rear power switch.
3. The heating element will begin to warm up.
4. Connect the controller to the manual-control socket.
5. Press the manual control button to spray fog.
6. The first spray may take a few seconds after warm-up.
7. If using the remote control, connect the receiver to the remote-control socket.
8. Use the remote buttons to trigger output.

Note:
- For first use, expect a short delay before the machine produces fog.

---

## 7. Usage Warnings

- Do not run the machine without fog fluid.
- Do not contaminate the fog fluid.
- Do not block the nozzle.
- Do not use the machine in wet or damp conditions.
- Use the correct input voltage only.
- The machine includes overheat protection, but regular maintenance is still recommended.
- Clean the machine with a dry cloth.
- Store the machine in a dry place.

---

## 8. Troubleshooting

### No Fog Output

1. Check whether the fog fluid tank is empty.
2. Check whether the nozzle is blocked.
3. Check whether the pump is drawing fluid correctly.
4. Check whether the sensor or thermostat is working properly.
5. Check whether the machine has finished warming up.

### Abnormal Condition or Suspected Fault

1. Disconnect the unit from mains power immediately.
2. Inspect the power connection and basic wiring.
3. If the problem continues, stop using the machine and have it serviced.
